For Journal 4 I chose Babe Ruth Central, a site that demonstrated the concepts outlined in chapter 6 of Steve Krug's book Don't Make Me Think. In this chapter, Krug emphasized the importance of a clear system of navigation.

}

'Babe Ruth Central's Purpose

Babe Ruth Central features a very effective "Primary Navigation" bar. Immediately upon accessing this site I noticed the sections entitled "Statistics" and "Biography". This led me to believe that the site serves as an informational site on Babe Ruth. Continuing along the navigation bar, I noticed "BR Store" which told me that this site has a webstore element.

Babe Ruth Central also has a series of "news items" that display briefly on the middle of the screen. Though not particularly effective (see my critique below), one of the elements is a "welcome" screen that introduces the site.

Critique

As far as interface design goes, Babe Ruth Central conforms to the web standards for navigation (as outlined by Krug). The site has a clear visual heirarchy, dedicating the top portion to navigation and splitting the rest of the page up into sections.

The site maintains a set of persistent navigational elements, a site ID at the top of every page reminds the user that they are still on the Babe Ruth Central web page.

The primary navigation bar features short, useful links to the site's main pages. The links visually change depending on which page you are on.

Additionally, the site features an interactive secondary navigation bar that appears when you hover over the primary navigation, a search bar, and bread crumbs on every page.

The rotating "news items" on the front page are largely unnecessary, especially the "welcome" page. This is what Krug would label as "Happy Talk". With such a clear design, users don't need any additional instruction to use the Babe Ruth Central site and a "welcome" page simply takes up space.

Unfortunately, the site is hardly usable. Clicking on "Biography" results in a "redirect loop" that prevents users from accessing the page, the "Babe's Impact" link redirects to the "Hero & Icon" page, etc. These issues would prevent the typical Internet user from frequenting this site.

Potential Improvements

The first thing that the developers of Babe Ruth Central MUST do is fix the technical issues preventing proper use of their site. Redirect loops and links directing to the wrong pages are easy fixes and their presence indicates lazy development.

On the design end, I would remove the rotating "news items" on the home page. Everything contained within is either self-explanatory or found elsewhere. If these items are absolutely necessary, they should be static elements and should take up much less of the front page.
