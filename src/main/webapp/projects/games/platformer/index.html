project(platformer) {

This was a game I developed for a telecommunications course at Indiana University.

The goal of this game was to adapt Gary Rosenzweig's platformer code from his book ActionScript 3.0 Game Programming University into a new project. In the end, I borrowed Rosenzweig's collision detection code but developed:
  • Arrow traps.
  • Ladder climbing.
  • Levers for doors.
  • Mouse-click to attack.
  • Stealth (crouch).
  • UI elements (checkboxes, popups).
  • ...And more.

This game uses Adobe Flash.

}
